# 職務経歴

## 概要

2019年5月〜正社員として、自社開発の占いサービスの新規開発・保守に携わった後、SESとしてWebシステムの開発・運用に従事。プロジェクトでは、後進メンバーへの知見共有やDocker環境の整備を通じて、  
バックエンド・フロントエンド・インフラを横断した技術支援を担当し、開発チーム間の連携強化とボトルネックとなっていたバグの解消に貢献しました。

2023年4月〜フリーランスとして、金融系・フードデリバリー・プログラミング学習サービスなどのWebサービス開発・保守に関わり、サブリーダーとしてタスクアサイン、バグ改善、コードレビューを担当。  
プロダクトの理解が深まるにつれ、チームの開発フロー改善や品質向上にも貢献しました。

AWSのサーバーレス技術（Lambda, API Gateway）、コンテナ技術（ECS, EKS）、ALBやEC2などの基盤技術も経験していますが、インフラ専任ではなく、Web開発者としてインフラにも関わる形でビジネスに貢献したいと考えています。

## 業務外活動

- 業務で利用する技術の検証環境を個人のGitHubリポジトリ、AWSアカウント内に構築し、素早くチームや事業に貢献できるように努めています。
- 個人で開発、検証した結果について[blog](https://dev63.hatenablog.com/)、[Qiita](https://qiita.com/dev63)、[Zenn](https://zenn.dev/dev63)に投稿しナレッジを残すようにしています。
- 直近ではOSSのコードリーディングを始めたり、1件ですが[textlint-ja](https://github.com/textlint-ja/textlint-rule-preset-JTF-style/pull/147)というOSSにコントリビュートしたりしています。
