2019 年 5 月よりソフトウェアエンジニアとして活動。  
6 年の実務経験があり、プロダクトの成長を最優先に、フロントエンド・バックエンド・インフラまで幅広く対応してきました。

リーダーやサブリーダーの経験があり、チーム横断の調整やビジネスサイドとの連携も行ってきました。
