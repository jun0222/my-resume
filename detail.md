# 職務経歴

## 概要

2019年5月〜正社員として、自社開発の占いサービスの新規開発・保守に携わった後、SESとしてWebシステムの開発・運用に従事。バックエンド・フロントエンド・インフラを横断した技術支援を通じて、開発チーム間の連携強化と重要課題の解決に貢献しました。

2023年4月〜フリーランスとして、金融系・フードデリバリー・プログラミング学習サービスなどのWebサービス開発・保守に関わり、サブリーダーとしてタスク管理からコードレビューまで幅広く担当。プロダクト理解を深めながら、開発フロー改善や品質向上にも積極的に取り組みました。

AWSのサーバーレス技術（Lambda, API Gateway）、コンテナ技術（ECS, EKS）、基盤技術（ALB, EC2）も経験していますが、インフラ専任ではなく、Web開発者としてインフラにも関わる形でビジネスに貢献したいと考えています。

## 業務外活動

- 業務利用技術の検証環境を個人のGitHubリポジトリ、AWSアカウント内に構築し、迅速なチーム貢献を実現
- 開発・検証結果を[blog](https://dev63.hatenablog.com/)、[Qiita](https://qiita.com/dev63)、[Zenn](https://zenn.dev/dev63)に投稿し知見を共有
- OSSコードリーディングや[textlint-ja](https://github.com/textlint-ja/textlint-rule-preset-JTF-style/pull/147)へのコントリビュートを実施
